基础语法
========

变量赋值
--------

::

    name = "PyStart"    # 字符串
    age = 18            # 整数
    pi = 3.14           # 浮点数
    is_ok = True        # 布尔值

多重赋值::

    a, b, c = 1, 2, 3
    x = y = z = 0


注释
----

::

    # 单行注释
    
    """
    多行注释
    可以写很多行
    """


输入输出
--------

::

    print("Hello, World!")           # 输出
    print("a =", a, "b =", b)        # 输出多个值
    print(f"name: {name}, age: {age}") # f-string格式化
    
    name = input("请输入姓名: ")      # 输入（返回字符串）
    age = int(input("请输入年龄: "))  # 输入并转为整数


运算符
------

**算术运算符**::

    +   加法      5 + 3 = 8
    -   减法      5 - 3 = 2
    *   乘法      5 * 3 = 15
    /   除法      5 / 3 = 1.666...
    //  整除      5 // 3 = 1
    %   取余      5 % 3 = 2
    **  幂运算    5 ** 3 = 125

**比较运算符**::

    ==  等于      a == b
    !=  不等于    a != b
    >   大于      a > b
    <   小于      a < b
    >=  大于等于  a >= b
    <=  小于等于  a <= b

**逻辑运算符**::

    and   与    a > 0 and b > 0
    or    或    a > 0 or b > 0
    not   非    not a > 0
